# Como ejecutar este código

Prerrequsitos :

-   Tener Nodejs y git instalados globalmente en su computadora
-   Clonar el repositorio de github y moverse a la rama ejercicios 20 de julio como se indica en el siguiente bloque

```bash
git clone https://github.com/stan-mqz/Programacion-Orientada-a-Eventos.git

cd /Programacion-Orientada-a-Eventos

git  git checkout ejercicios_20_julio 

```

1.  Instalar las dependencias del proyecto

```bash
npm install
```

2.  Ejecutar el siguiente comando en la consola

```bash
#Ejemplo: npx tsx ./src/ejercicio1.ts
npx tsx ./src/${nombre_archivo}

```
